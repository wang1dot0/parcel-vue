# parcel-vue
Use parcel to build a vue'js app.
Functions: vue-router, vuex, less, babelrc, eslint, HMR & express.

## How to use
```
# install dependency
npm install

# run HMR
npm run dev

# run express server
npm run start

# production mode
npm run build

# code eslint
npm run lint              
```

**Cannot support IE < 11 becauseof parcel not polyfill `Object.assign`**
